# Workleto - Invoice app

## How to run 

1. Clone the repository 
    - git clone https://github.com/ilcand/Workleto.git
    - cd Workleto

2. Install dependencies 
    - composer install 

3. Build required docker images 
    - docker-compose up -d --build

4. Application runs on port 8080
    - http://localhost:8080
